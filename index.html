conrich 

conrich為康和綜合證券程式下單API，整合了.net framework 4.5架構下的交易介面，讓使用者可以輕鬆的運用python 3.6+ 執行快速下單。

conrich is a securities, futures, option API, which send orders to Concord Securities Group. This package integrated with .net framework 4.5, and is compatible with Python 3.6+. It is distributed under the MIT license. All functions which offer by conrich is summarized in the chart
